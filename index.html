☁️ GitHub Gist Sync

Sync your data across devices using a GitHub Gist. Free, private, and persistent.

How to create your token:
1. Go to github.com/settings/tokens/new
2. Select "Tokens (classic)" (NOT Fine-grained)
3. Enter a note like "Parcel Quest"
4. Set expiration (e.g. 90 days)
5. ✅ Check only the "gist" scope box
6. Click Generate token at the bottom
7. Copy the token immediately (you won't see it again!)
8. Paste it below
